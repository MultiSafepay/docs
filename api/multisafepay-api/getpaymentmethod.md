## How to test

1. Under **Authentication** > **Bearer**, enter an [API key](/docs/sites#site-id-api-key-and-security-code) from your **TEST** account.
2. Under **Path params**, enter the payment method ID.
3. Click **Try it!**

❗️ Never use a live API key!
