## How to test

1. Under **Authentication** > **Query**, enter an [API key](/sites#site-id-api-key-and-security-code) from your **TEST** account.
2. Under **Path params**, enter the gateway ID.
3. Click **Try it!**

❗️ Never use a live API key!
