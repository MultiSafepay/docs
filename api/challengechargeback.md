## How to test

1. Under **Authentication** > **Query**, enter an [API key](/websites#site-id-api-key-and-security-code) from your **TEST** account.
2. Under **Path params**, enter the order ID of the order you want to challenge a chargeback for.
3. Under **Body params**, fill in the required parameters.
4. Click **Try it!**

:warning: Never use a live API key!
