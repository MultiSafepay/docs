This technical manual is for installing and configuring our free app for integrating MultiSafepay payment solutions into your LogiVert app.

{{< details title="Test environment" >}}
&nbsp;  
We recommend first installing the app in a test environment following the recommended LogiVert installation procedure. Make sure you have made a backup.
{{< /details >}}

{{< details title="Support" >}}
&nbsp;  
For technical support, contact LogiVert - [Support](https://www.logivert.com/nl/support/c-10).

{{< /details >}}

{{< details title="Requirements" >}}
&nbsp; 
You will need a MultiSafepay account. See [Getting started](/getting-started/).

{{< /details >}}

For installation and configuration instructions, see LogiVert - [Manual](https://confluence.prezent.nl/display/LOGIVERTMAN/Het+specificeren+van+betalingswijzen).
