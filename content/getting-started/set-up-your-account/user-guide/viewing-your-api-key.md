To view your site ID, API key, and secure code for a specific website:

1. Sign in to your [live](https://merchant.multisafepay.com) or [test](https://testmerchant.multisafepay.com) MultiSafepay account.

2. Go to **Settings** > **Website settings**.

3. Click on the relvant website to view the **Website details** page.
