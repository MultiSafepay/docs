## Gateway issuers

**Parameters**

----------------

__code__ | string

The unique identifier of the payment gateway to retrieve an issuer list for. Supported identifiers are: [iDEAL](#ideal-issuers) 

----------------

Direct iDEAL transactions can have the following issuers:

| IssuerID | Bank                  |
|----------|-----------------------|
| 0031     | ABN AMRO              |
| 0761     | ASN Bank              |
| 4371     | bunq                  |
| 1235     | Handelsbanken         |
| 0721     | ING                   |
| 0801     | Knab                  |
| 0021     | Rabobank              |
| 0771     | Regio Bank            |
| 1099     | Revolut               |
